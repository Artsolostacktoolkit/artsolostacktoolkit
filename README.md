# SoloStackToolkit — Разработчик кроссплатформенного ПО 🖥️

### Экспертиза: Rust · Tauri · WPF · UI/UX · A11y

Привет! Я специализируюсь на создании современного кроссплатформенного программного обеспечения (Windows, macOS, Linux). Мое главное направление — **Rust в связке с Tauri**, позволяющее создавать легковесные, безопасные и производительные приложения. Также владею WPF для задач, где требуется глубокая интеграция с экосистемой Windows.

В отличие от классического фриланса «пишу код на заказ», я предлагаю **готовые инструменты**, которые уже решают бизнес-задачи. Мои программы можно внедрить сразу или быстро адаптировать под конкретные нужды.

Меня влечет создание **брендированных, нестандартных интерфейсов**, которые выделяют продукт среди конкурентов. Я не просто верстаю окна — я проектирую взаимодействие:

- 🔹 Разработка **кастомных контролов** под уникальные задачи бизнеса
- 🔹 Создание **адаптивных интерфейсов**, работающих на любых разрешениях
- 🔹 Соблюдение стандартов **доступности (A11y)** — интерфейсы, удобные для всех пользователей
- 🔹 Следование принципам **инклюзивного дизайна (ID)**

Каждое приложение, которое я создаю, не только функционально, но и приятно в использовании.
